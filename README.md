## How to install the WordPress theme

1. Open your WordPress dashboard.
2. Go to Appearance > Themes > Add New > Upload Theme.
3. Zip the folder `wordpress-theme/piar-research-university`.
4. Upload and activate it.
5. Create a page called `Home`.
6. Go to Settings > Reading and set the homepage to that page.
7. Create a primary menu and assign it under Appearance > Menus.
